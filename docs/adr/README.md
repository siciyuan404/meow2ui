# ADR Guide

Use Architecture Decision Records (ADR) to document major technical decisions.

## Naming

- File format: `NNNN-short-title.md`
- Example: `0001-flow-orchestration-runtime.md`

## Lifecycle

- Proposed
- Accepted
- Superseded
- Deprecated

## Required Sections

- Context
- Decision
- Consequences
- Alternatives

Start from `docs/adr/0000-template.md`.
